Drag and Drop

Basic Draggables

Use clrDraggable directive to make an element or component draggable.

Custom Ghosts

You can use another element or component as a draggable ghost using clr-draggable-ghost component with *clrIfDragged.

Draggable Handle

Sometimes there are cases where users should be able to move a whole element just by dragging with a handle element. To do that, you can use clrDraggableHandle directive.
